Términos de Uso

  • 1. Aceptación: Al usar el bot, aceptas estos términos y las directrices de Discord.
  • 2. Uso Prohibido: No se permite el uso del bot para spam, flood o ataques masivos.
  • 3. Integridad: Queda prohibido intentar extraer el código fuente o explotar bugs.
  • 4. Responsabilidad: No nos hacemos responsables por la mala configuración del bot en tu servidor.
  • 5. Contenido: El usuario es responsable del contenido generado mediante comandos de texto.
  • 6. Suspensión: Nos reservamos el derecho de banear servidores que incumplan las normas.
  • 7. Modificaciones: Los términos pueden cambiar; el uso continuo implica aceptación.
  • 8. Gratuidad: El bot es gratuito, pero algunas funciones premium pueden requerir donación.

Política de Privacidad

  • 1. Datos Recopilados: Almacenamos IDs de usuario, de servidor y de canales de texto.
  • 2. Finalidad: Los datos solo sirven para sistemas de economía, niveles y logs.
  • 3. Transparencia: No vendemos ni compartimos información con empresas externas.
  • 4. Seguridad: Los datos se guardan en bases de datos cifradas y protegidas.
  • 5. Mensajería: El bot no lee mensajes privados (DMs) a menos que sean comandos.
  • 6. Retención: Los datos se borran si el bot es expulsado del servidor tras 30 días.
  • 7. Derechos: Puedes solicitar una copia de tus datos a través de nuestro soporte.
  • 8. Eliminación: Usa los comandos de "borrado de perfil" para limpiar tu información.
