Términos y Privacidad

1. Términos de Servicio

Al usar este Bot, aceptas cumplir con las reglas de Discord. El bot se ofrece "tal cual" para mejorar tu servidor.

  • No usar el bot para spam.
  • Respetar la privacidad de otros usuarios.

2. Política de Privacidad

Solo guardamos tu ID de Discord y configuraciones del servidor para que el bot funcione. No vendemos tus datos a nadie.

Eliminación: Puedes pedir borrar tus datos saliendo del servidor o contactando al soporte.
